# 利用广播实现强制下线功能

在界面中点击Button，弹出一个对话框，让用户无法进行任何操作，必须点击对话框中的确定按钮，然后回到登录界面重新登录。

## 模拟登录界面
在登录界面，初始账号和密码为 admin 和 123456,当用户输入正确的账号密码，则进入广播界面，如输入错误，则弹窗提示
